# 5.2.25 RELEASE UNIT (6/10)- 17h/57h

The Release Unit command is defined in SPC-2 (see http://www.t10.org/cgi-bin/ac.pl?t=f&f=spc2r20.pdf). This
clause decribes the specific implementation.

```
                                       Table 106 — RELEASE UNIT (6) CDB

                                                                   Bit
       Byte
                    7 msb         6            5             4              3              2          1        0 lsb
        0                                               OPERATION CODE (17h)

        1                     Obsolete                                             Obsolete (00h)
        2                                                        Obsolete
        3
                                                                 Reserved
        4
        5                                             Control Byte (see 5.1.2.3)
```

The following parameters apply:
- **Obsolete**: All obsolete fields shall be set to zero.

```
                                      Table 107 — RELEASE UNIT (10) CDB

                                                                   Bit
       Byte
                    7 msb         6            5             4              3              2          1        0 lsb
        0                                               OPERATION CODE (57h)

        1                     Reserved                   3RDPTY                 Reserved            LONGID    Obsolete
        2                                                        Obsolete
        3                                               THIRD PARTY DEVICE ID

        4                                                        Reserved
        5                                                        Reserved
        6                                                        Reserved
        7
                                                        PARAMETER LIST LENGTH
        8
        9                                       Control Byte (see 5.1.2.3)
```

The following parameters apply:
- **Obsolete**: All obsolete fields shall be set to zero.
- **LONGID**: 0b
- **3RDPTY**: 0b
- **THIRD PARTY DEVICE ID**: 00h

- **PARAMETER LIST LENGTH**: 0000h
