# 6.3.15 IP C1h: Drive Serial Numbers

- **INQUIRY - 12h** (see 5.2.5 on page 84) describes how to request this page.

## 6.3.15.1 Returned Data - IP C1h: Drive Serial Numbers

For LUN 0, the following data is returned:

> WARNING: The length of this page should be dynamically parsed. The size of this page has increased in the past and may increase in the future (i.e., additional fields may be added).

```
Table 222 — IP C1h: Drive Serial Numbers

                                                Bit
Byte    7           6           5           4           3           2           1       0
  0     Peripheral Qualifier (000b)                     Peripheral Device Type (01h)
  1                                 Page Code (C1h)
  2                               Reserved
  3                           Page Length (24h)
  4
                Manufacturing Full Assembly Serial Number
  15
  16
                  Reported Full Assembly Serial Number
  27
  28
                Manufacturing Drive Brick Serial Number
  39
```

- **Byte 0** - Peripheral Data:
  - Bits 7-5: Peripheral Qualifier: 000b
  - Bits 4-0: Peripheral Device Type: 01h
- **Byte 1** - Page Code: C1h
- **Byte 2** - Reserved
- **Byte 3** - Page Length: 24h
- **Bytes 4-15** - Manufacturing Full Assembly Serial Number: Full Assembly Serial Number set at time of manufacture, right-justified with leading zeros, in ASCII.
- **Bytes 16-27** - Reported Full Assembly Serial Number: Full Assembly Serial Number used as the serial number in the Unit Serial Number Inquiry page (i.e., page 80h), right-justified with leading zeros, in ASCII. This value may be over-ridden by a library or FRU process.
- **Bytes 28-39** - Manufacturing Drive Brick Serial Number: This is the drive brick serial number set at time of manufacture, right-justified with leading zeros, in ASCII.
