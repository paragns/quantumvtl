# 6.1.11 Diag - 0161h: Write Dump to Cartridge

This diagnostic causes dump information residing in the device control storage to be written to a cartridge without the need to retrieve the dump data across the host interface. A cartridge is required to be loaded to run the diagnostic. See 5.2.36--SEND DIAGNOSTIC - 1Dh and 5.2.23--RECEIVE DIAGNOSTIC RESULTS - 1Ch for additional information on the commands.
