# 6.6.6 MP 02h: Disconnect-Reconnect

See 5.2.10 for how to set these parameters and 5.2.11 for how to read these parameters. Mode Page Behaviors (see 4.7 on page 40) describes this device's non-standard behaviors related to mode parameters.

The mode page policy for this page is described in 4.7.1.


- **Byte 0**:
  - Bit 7: **PS** (Parameter Saveable): 0b. Ignored in MODE SELECT. See 4.7.2.2.
  - Bit 6: Reserved
  - Bits 5-0: **PAGE CODE**: 02h
- **Byte 1**: PAGE LENGTH: 0Eh
- **Byte 2**: BUFFER FULL RATIO: 00h (non-changeable)
- **Byte 3**: BUFFER EMPTY RATIO: 00h (non-changeable)
- **Bytes 4-5**: BUS INACTIVITY LIMIT: (non-changeable)
- **Bytes 6-7**: DISCONNECT TIME LIMIT: 0000h (no limit) (non-changeable)
- **Bytes 8-9**: CONNECT TIME LIMIT: (non-changeable)
- **Bytes 10-11**: MAXIMUM BURST SIZE: 0000h (no limit) (changeable)
  - This value is changeable and remembered, but not used.
- **Byte 12**:
  - Bit 7: **EMDP** (Enable Modify Data Pointers): 0b (non-changeable)
  - Bits 6-4: **FAIR ARBITRATION**: 000b (non-changeable)
  - Bit 3: **DIMM** (Disconnect Immediate): 0b (non-changeable)
  - Bits 2-0: **DTDC** (Data Transfer Disconnect Word): 000b (non-changeable)
- **Byte 13**: Reserved
- **Bytes 14-15**: FIRST BURST SIZE: 0000h (non-changeable)
