# 6.3.10 IP B1h: Manufacturer-assigned Serial Number

INQUIRY - 12h (see 5.2.5 on page 84) describes how to request this page. The following parameters apply to this request:

- **EVPD** (Enable Vital Product Data): 1b
- **Page Code**: B1h
- **Allocation Length**: 0Eh (14) bytes available

## 6.3.10.1 Returned Data - IP B1h: Manufacturer-assigned Serial Number

Table 216 specifies the Manufacturer-assigned Serial Number VPD page.

```
Table 216 — IP B1h Manufacturer-assigned Serial Number VPD page

                                                Bit
Byte    7           6           5           4           3           2           1       0
  0     PERIPHERAL QUALIFIER (000b)                     PERIPHERAL DEVICE TYPE (01h)
  1                                 PAGE CODE (B1h)
  2     (MSB)
                            PAGE LENGTH (000Ah)
  3                                                                                     (LSB)
  4     (MSB)
                    MANUFACTURER-ASSIGNED SERIAL NUMBER
  13                                                                                    (LSB)
```

- **Byte 0** - Peripheral Data:
  - Bits 7-5: PERIPHERAL QUALIFIER: 000b
  - Bits 4-0: PERIPHERAL DEVICE TYPE: 01h
- **Byte 1** - PAGE CODE: B1h
- **Bytes 2-3** - PAGE LENGTH: 0Ah
- **Bytes 4-13** - MANUFACTURER-ASSIGNED SERIAL NUMBER: Right-aligned ASCII data that is the manufacturer-assigned serial number. If the manufacturer-assigned serial number is not available, the device returns ASCII spaces (20h) in this field. If the manufacturer-assigned serial number differs from the value in the UNIT SERIAL NUMBER field of IP 80h: Unit Serial Number (see 6.2.3), then the value in the UNIT SERIAL NUMBER field is used in building the T10 vendor ID descriptor.
