# 6.4.27 LP 39h[02h]: Host Port 0 Physical Interface

See 5.2.8--LOG SELECT - 4Ch and see 5.2.9--LOG SENSE - 4Dh for directions on how to use this page. Physical information about host interface port 0. This page is only returned on Fibre Channel drives.

## 6.4.27.1 Parameter Definitions (39h[02h])

> **WARNING:** Log pages may have different reset behaviors. Care should be taken to ensure counters are processed appropriately, including consideration of their reset behaviors. While reset behaviors may be different between products, an attempt is made to maintain compatibility between generations of a given product. See 6.4--Log Parameters (LP)

> **WARNING:** The parameters in log pages should be dynamically parsed as some parameters may not be present and new parameters may be inserted. The relative location of parameters have changed and are anticipated to continue to change.

| Code | Counter: Description | Size |
|------|----------------------|------|
| 0000h | **Host Port SFF-8472 Address A2h {LP39h[02h]:0000h}**: See 6.4.27.1.1 | 132 |

### 6.4.27.1.1 Host Port SFF-8472 Address A2h

> **WARNING:** The information from the SFP is updated on a periodic basis; No time since last update is provided.

```
                                              Bit
Byte
         7 msb      6         5         4         3         2         1      0 lsb
  0      (MSB)
                                        PARAMETER CODE
  1                                                                          (LSB)
                          Parameter list control byte - binary format list log parameter
  2
           DU    Obsolete    TSD                Obsolete               FORMAT AND LINKING

  3                                  PARAMETER LENGTH (84h)

  4                               PHYSICAL INFORMATION FORMAT

  5
  6                                        Reserved
  7
  8
                              Host Port SFF-8472 Address A2h data
135
```

- **Bytes 0-1** PARAMETER CODE: 0000h
- **Byte 2** Parameter list control byte - binary format list log parameter
  - Bit 7: DU: 0b
  - Bit 6: Obsolete
  - Bit 5: TSD: 0b
  - Bits 4 to 2: Obsolete
  - Bits 1 to 0: FORMAT AND LINKING: 11b
- **Byte 3** PARAMETER LENGTH: (84h)
- **Byte 4** PHYSICAL INFORMATION FORMAT: The format of the DT device primary port physical interface information data
  - 00h: No information is available. The DT device primary port physical interface information data is not valid.
  - 02h: The Host Port SFF-8472 Address A2h data is set to bytes 0 through 127 of the SFF-8472 physical interface memory map address A2h.
  - others: Reserved.
- **Bytes 5 to 7** Reserved
- **Bytes 8 to 135** Host Port SFF-8472 Address A2h data (see the PHYSICAL INFORMATION FORMAT field).
