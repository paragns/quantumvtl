# 6.3.13 IP B5h: Logical Block Protection

INQUIRY - 12h (see 5.2.5 on page 84) describes how to request this page.

- **Byte 0** - Peripheral Data:
  - Bits 7-5: PERIPHERAL QUALIFIER: 000b
  - Bits 4-0: PERIPHERAL DEVICE TYPE: 01h
- **Byte 1** - PAGE CODE: B5h
- **Bytes 2-3** - PAGE LENGTH: n-3
- **Bytes 4-7** - Reserved
- **Bytes 8-n** - The logical block protection method descriptor list contains a list of descriptors which describe each supported logical block protection method. The descriptors are returned in ascending order by logical block protection method code starting with logical block protection method zero (i.e., LBP not enabled). The list of logical block protection method descriptors depends on the product.

  For x=1 to the number of descriptors:

  - **Byte 8x+0** - LOGICAL BLOCK PROTECTION METHOD DESCRIPTOR LENGTH: 07h
  - **Byte 8x+1** - LOGICAL BLOCK PROTECTION METHOD
  - **Byte 8x+2**:
    - Bits 7-6: Reserved
    - Bits 5-0: LOGICAL BLOCK PROTECTION INFORMATION LENGTH
  - **Byte 8x+3**:
    - Bit 7: LBP_W_C (logical blocks protected during write capable)
    - Bit 6: LBP_R_C (logical block protected during read capable)
    - Bit 5: RBDP_C (recover buffered data protected capable)
    - Bits 4-0: Reserved
  - **Bytes (8x+4) to (8x+7)** - Reserved
