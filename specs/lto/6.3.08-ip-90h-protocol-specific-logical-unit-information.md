# 6.3.8 IP 90h: Protocol-Specific Logical Unit Information

The Protocol-Specific Logical Unit Information VPD page is returned on LUN 0 of SAS attached devices that support changing the TLR control field in the frame header and contains parameters for logical unit 0 that are protocol-specific based on the I_T nexus being used to access the logical unit.

- **INQUIRY - 12h** (see 5.2.5 on page 84) describes how to request this page.

## 6.3.8.1 Returned Data - IP 90h: Protocol-Specific Logical Unit Information

Table 213 defines the Protocol-Specific Logical Unit Information VPD page for logical units with SAS target ports.

```
Table 213 — IP 90h Protocol-Specific Logical Unit Information VPD page for SAS SSP

                                                    Bit
Byte    7           6           5           4           3           2           1       0
  0     PERIPHERAL QUALIFIER (000b)                     PERIPHERAL DEVICE TYPE (01h)
  1                                 PAGE CODE (90h)
  2     (MSB)
                                PAGE LENGTH (18h)
  3                                                                                     (LSB)
                    Logical unit information descriptor list
  4
        Logical unit information descriptor (first)(see table 214)
  15
  16
        Logical unit information descriptor (second)(see table 214)
  27
```

- **Byte 0** - Peripheral Data:
  - Bits 7-5: PERIPHERAL QUALIFIER: 000b
  - Bits 4-0: PERIPHERAL DEVICE TYPE: 01h
- **Byte 1** - PAGE CODE: 90h
- **Bytes 2-3** - Page Length: 18h
- **Bytes 4-27** - Logical unit information descriptor list: The logical unit information descriptor list contains a logical unit information descriptor for each SAS target port known to the device server. See 6.3.8.2--Logical unit information descriptor.

## 6.3.8.2 Logical unit information descriptor

Table 214 defines the logical unit information descriptor for logical units with SAS target ports.

```
Table 214 — Logical unit information descriptor for SAS SSP of IP 90h

                                                    Bit
Byte    7           6           5           4           3           2           1       0
  0     (MSB)
                        RELATIVE PORT IDENTIFIER
  1                                                                                     (LSB)
  2             Reserved                                    PROTOCOL IDENTIFIER (6h)
  3
                                Reserved
  5
  6     (MSB)
                        DESCRIPTOR LENGTH (0004h)
  7                                                                                     (LSB)
                Per logical unit SCSI transport specific data
  8                         Reserved                                                    TLR CTL SUP
  9
                                Reserved
  11
```

- **Bytes 0-1** - RELATIVE PORT IDENTIFIER: The Relative Target Port from the IP 83h: Device Identification (see 6.3.4 on page 230).
- **Byte 2**:
  - Bits 7-4: Reserved
  - Bits 3-0: PROTOCOL IDENTIFIER: 06h (i.e. SAS SSP specific descriptor)
- **Bytes 3-5** - Reserved
- **Bytes 6-7** - DESCRIPTOR LENGTH: 0004h
- **Byte 8**:
  - Bits 7-1: Reserved
  - Bit 0: TLR CTL SUP (TLR Control Supported):

    | Value | Description |
    |-------|------------|
    | 0b | The combination of the SCSI target port and logical unit do not support the TLR CONTROL field in the SSP frame header. |
    | 1b | The combination of the SCSI target port and logical unit support the TLR CONTROL field in the SSP frame header. |

- **Bytes 9-11** - Reserved
