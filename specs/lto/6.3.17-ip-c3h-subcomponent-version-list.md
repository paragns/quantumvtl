# 6.3.17 IP C3h: Subcomponent Version List

- **INQUIRY - 12h** (see 5.2.5 on page 84) describes how to request this page.

## 6.3.17.1 Returned Data - IP C3h: Subcomponent Version List

For LUN 0, the following data is returned:

> WARNING: This page should be dynamically parsed. Additional subcomponents (i.e., fields) may be added in the future.

```
Table 226 — IP C3h: Subcomponent Version List

                                                Bit
Byte    7           6           5           4           3           2           1       0
  0     PERIPHERAL QUALIFIER (000b)                     PERIPHERAL DEVICE TYPE (01h)
  1                                 PAGE CODE (C3h)
  2     (MSB)
                                PAGE LENGTH (n-3)
  3                                                                                     (LSB)
                    Subcomponent version descriptor list
  4
            Subcomponent version descriptor (first)
                                    ...
            Subcomponent version descriptor (last)
  n
```

- **Byte 0** - Peripheral Data:
  - Bits 7-5: PERIPHERAL QUALIFIER: 000b
  - Bits 4-0: PERIPHERAL DEVICE TYPE: 01h
- **Byte 1** - PAGE CODE: C3h
- **Bytes 2-3** - PAGE LENGTH
- **Bytes 4-n** - Subcomponent version descriptor list: Each Subcomponent version descriptor (see 6.3.17.1.1) describes a Subcomponent version returned by the drive.

### 6.3.17.1.1 Subcomponent version descriptor

Descriptor that describes a Subcomponent version.

```
Table 227 — Subcomponent version descriptor

                                                Bit
Byte    7           6           5           4           3           2           1       0
  0
                        SUBCOMPONENT DESIGNATOR
  1
  2                     SUBCOMPONENT LENGTH (m-2)
  3
                        SUBCOMPONENT VERSION
  m
```

- **Bytes 0-1** - SUBCOMPONENT DESIGNATOR: A value that designates which subcomponent version is contained in the SUBCOMPONENT VERSION field.
- **Byte 2** - SUBCOMPONENT LENGTH: Number of bytes to follow. Depends on the value in the SUBCOMPONENT VERSION DESIGNATOR field.
- **Bytes 3-m** - SUBCOMPONENT VERSION: The version of the subcomponent.

| SUBCOMPONENT DESIGNATOR | SUBCOMPONENT LENGTH | Description |
|------------------------|--------------------|----|
| 0000h | 01h | Dynamic braking (e.g., GDB, EDB) version from the PLD |
