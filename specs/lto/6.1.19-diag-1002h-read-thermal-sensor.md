# 6.1.19 Diag - 1002h: Read Thermal Sensor

The Read Thermal Sensor diagnostic is used to read the digital thermal sensor on the drive. See 5.2.36--SEND DIAGNOSTIC - 1Dh and 5.2.23--RECEIVE DIAGNOSTIC RESULTS - 1Ch for additional information on the commands.

Starting in LTO-9 it is better to get the temperature from LP 0Dh[01h]: Environmental Reporting (LTO9 and later) (see 6.4.9) where the humidity is also reported.

Table 184 shows the parameter data for the Read Thermal Sensor diagnostic.

```

```

## 6.1.19.2 RcvDiag Data - Read Thermal Sensor

Table 185 shows the diagnostic results data received from the Read Thermal Sensor diagnostic.

```
Receive Diag parameter data - Read Thermal Sensor (Table 185)

  Byte    7     6     5     4     3     2     1     0
    0                    PAGE CODE (80h)
    1                       Reserved
    2   (MSB)
                     PAGE LENGTH (000Eh)
    3                                              (LSB)
    4   (MSB)
                   DIAGNOSTIC ID (1002h)
    5                                              (LSB)
    6                FLAGS (0000000b)
    7                       Reserved
    8            FLAGS (00000b)       DIAG.  Reserved  ERROR
                                    BLOCKED
    9                       Reserved
   10       THERMAL READING (units of degC) (a)
   11                       Reserved
   12       Full-Height: WARNING THRESHOLD (units of degC) (a)(b)
            Half-Height: FAN START TEMPERATURE (units of degC) (a)(b)
   13                       Reserved
   14       FENCING THRESHOLD (units of degC) (a)
   15                       Reserved
   16       FENCING REMOVAL THRESHOLD (units of degC) (a)
   17                       Reserved
```

> (a) A value of 00h indicates the value is not applicable.

> (b) How to determine if a drive is a Full-Height drive or Half-Height drive is shown in Table 43, Standard Inquiry Product Identification Table.

- **PAGE CODE** (byte 0): 80h
- **Reserved** (byte 1)
- **PAGE LENGTH** (bytes 2-3): 000Eh
- **DIAGNOSTIC ID** (bytes 4-5): 1002h
- **FLAGS** (byte 6): 0000000b
- **Reserved** (byte 7)
- **Byte 8**: FLAGS -- DIAG. BLOCKED, Reserved, ERROR
- **Reserved** (byte 9)
- **THERMAL READING** (byte 10): Value read by sensor (units of degrees C)
- **Reserved** (byte 11)
- **Byte 12**:
  - Full-Height Drives: WARNING THRESHOLD (units of degrees C) -- drive raises an alert
  - Half-Height Drives: FAN START TEMPERATURE (units of degrees C)
- **Reserved** (byte 13)
- **FENCING THRESHOLD** (byte 14): Operations fenced (units of degrees C)
- **Reserved** (byte 15)
- **FENCING REMOVAL THRESHOLD** (byte 16): Remove operation fence (units of degrees C)
- **Reserved** (byte 17)
