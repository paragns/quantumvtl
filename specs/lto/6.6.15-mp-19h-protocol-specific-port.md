# 6.6.15 MP 19h: Protocol Specific Port

See 5.2.10 for how to set these parameters and 5.2.11 for how to read these parameters. Mode Page Behaviors (see 4.7 on page 40) describes this device's non-standard behaviors related to mode parameters.

The mode page policy for this page is described in 4.7.1.

The Protocol specific port mode page for FCP attached devices is defined in 6.6.15.1.

The Protocol specific port mode page for SAS attached devices is defined in 6.6.15.2.

## 6.6.15.1 MP 19h: FCP port

```
Table 344 — MP 19h Fibre Channel Port mode page

                                                          Bit
Byte
            7          6         5         4         3         2         1      0
  0       PS (0)   SPF (0)                        PAGE CODE (19h)

  1                                   PAGE LENGTH

  2                       Reserved                           PROTOCOL IDENTIFIER

  3      DTFD (0) PLPB (0) DDIS (0)  DLM (0)    RHA (0)  ALWI (0)  DTIPE (0) DTOLI (0)

  4
                                        Reserved
  5
  6                          Reserved                                RR_TOV UNITS

  7                       Resource Recovery Time Out Value (RR_TOV)
```

- **Byte 0**:
  - Bit 7: **PS** (Parameter Saveable): 0b. Ignored in MODE SELECT. See 4.7.2.2.
  - Bit 6: **SPF** (SubPage Format): 0b
  - Bits 5-0: **PAGE CODE**: 19h
- **Byte 1**: PAGE LENGTH: 06h

  > NOTE 63 - A PAGE LENGTH of 0Eh is allowed on Mode Select for backwards compatibility with older devices and levels of the standard. Any fields beyond byte 7 are (changeable-ignored).

- **Byte 2**:
  - Bits 7-1: Reserved
  - Bits 3-0: **PROTOCOL IDENTIFIER**: 0h (FCP) (non-changeable)
- **Byte 3**:
  - Bit 7: **DTFD** (Disable Target Fabric Discovery): 0b (non-changeable)
  - Bit 6: **PLPB** (Prevent Loop Port Bypass): 0b (non-changeable)
  - Bit 5: **DDIS** (Disable Discovery): 0b (non-changeable)
  - Bit 4: **DLM** (Disable Loop Master): 0b (non-changeable)
  - Bit 3: **RHA** (Require Hard Address): 0b (non-changeable)
  - Bit 2: **ALWI** (Allow Login without Loop Initialization): 0b (non-changeable)
  - Bit 1: **DTIPE** (Disable Target Initiated Port Enable): 0b (non-changeable)
  - Bit 0: **DTOLI** (Disable Target Originated Loop Initialization): 0b (non-changeable)
- **Bytes 4-5**: Reserved
- **Byte 6**:
  - Bits 7-3: Reserved
  - Bits 2-0: **RR_TOV UNITS** (changeable)

    | Value | Description |
    |-------|-------------|
    | 000b | No timer is specified: Default value is used and the command returns a CHECK CONDITION with Recovered Error, PARAMETERS ROUNDED (1/3700) |
    | 001b | Timer is specified in .001 second units |
    | 011b | Timer is specified in .1 second units |
    | 101b | Timer is specified in 10 second units |
    | others | Reserved |

- **Byte 7**: RR_TOV (Resource Recovery Time Out Value): (changeable)

  > NOTE 64 - The default RR_TOV value is 25 seconds.

- **Byte 0**:
  - Bit 7: **PS** (Parameter Saveable): 0b. Ignored in MODE SELECT. See 4.7.2.2.
  - Bit 6: **SPF** (SubPage Format): 0b (non-changeable)
  - Bits 5-0: **PAGE CODE**: 19h
- **Byte 1**: PAGE LENGTH: 0Eh
- **Byte 2**:
  - Bit 7: Reserved
  - Bit 6: **CONTINUE AWT** (continue arbitration wait timer): 1b (non-changeable)
  - Bit 5: **BRDCST ASYNCH EVENT**: 0b (non-changeable)
  - Bit 4: **RLM** (Ready LED Meaning): 0b (non-changeable)
  - Bits 3-0: **PROTOCOL IDENTIFIER**: 6h (SAS)
- **Byte 3**: Reserved
- **Bytes 4-5**: I_T NEXUS LOSS TIME: 07D0h (non-changeable)
- **Bytes 6-7**: INITIATOR RESPONSE TIMEOUT: 07D0h (non-changeable)
- **Bytes 8-9**: REJECT TO OPEN LIMIT: 0000h (non-changeable)
- **Bytes 10-15**: Reserved
