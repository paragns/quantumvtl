# 6.1.16 Diag - 0190h: Set Traps

This diagnostic permits a SCSI interface user to set a microcode trap that causes a dump to occur when the specified Fault Symptom Code occurs. The drive continues to operate after the dump completes. See 5.2.36--SEND DIAGNOSTIC - 1Dh and 5.2.23--RECEIVE DIAGNOSTIC RESULTS - 1Ch for additional information on the commands.
