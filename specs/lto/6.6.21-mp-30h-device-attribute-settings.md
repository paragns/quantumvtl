# 6.6.21 MP 30h: Device Attribute Settings

The device attribute settings page and the related subpages are used to query and configure settings used by the drive to control its behavior and configuration in a specified environment. MP 30h: Directory Listing - Device Attribute Settings (see 6.6.21.1 on page 404) is the directory listing of supported subpages and is in the table 331, Mode Page Layout, on page 368. Each subpage is for the query and/or setting of device attributes for a specific function and is in the Mode Page Subpage Layout (see Table 332 on page 369). The persistence of parameters and statement of which parameters are settable or only readable is stated in the section describing each subpage.

See 5.2.10 for how to set these parameters and 5.2.11 for how to read these parameters.

Mode Page Behaviors (see 4.7 on page 40) describes this device's non-standard behaviors related to mode parameters.

The mode page policy for this page is described in Mode Page Policy -- non-standard (see 4.7.1 on page 40).

#### 6.6.21.5.4.1 Preferred cartridge descriptor

A 4-byte tuple that defines a preferred cartridge type.

```
Table 365 — Preferred cartridge descriptor layout

                                                          Bit
Byte
            7          6         5         4         3         2         1      0
  0                                   MEDIUM TYPE

  1                                PREFERRED DENSITY

  2
                                        Reserved
  3
```

- **Byte 0**: MEDIUM TYPE: (non-changeable) A value indicating the physical cartridge type (see mode parameter header).
- **Byte 1**: PREFERRED DENSITY: (changeable-saveable) The density preferred for use with a Type M eligible cartridge (see 2.1.256), if:
  - C) initialization is performed on the cartridge;
  - D) a destructive overwrite is performed. For example:
    - a) a format operation is performed on the cartridge; or
    - b) logical object identifier zero of BOP 0 is written to the cartridge and the volume is a single partition volume.

  > NOTE 69 - The preferred cartridge setting is only used when a cartridge is Type M eligible (see 2.1.256), the cartridge is a Type A cartridge (see 2.1.254). Regardless of the preferred cartridge setting, a Type M cartridge (see 2.1.255) will not be changed back to a Type A cartridge.

  | Value | Description |
  |-------|-------------|
  | 00h | Use native cartridge density (e.g., Ultrium 7 cartridge use "Ultrium-7" density (see table 112, Density information LTO-7 and later, on page 142) |
  | others | If supported density, set to that density. If not a supported density, terminate the command with CHECK CONDITION status, with sense key set to ILLEGAL REQUEST, with additional sense code set to INVALID FIELD IN PARAMETER LIST (5/2600). |

- **Bytes 2-3**: Reserved

#### 6.6.21.5.4.2 Preferred Cartridge Type support

The medium type--preferred density pairs that are supported are shown in table 366.

| Product | MEDIUM TYPE | PREFERRED DENSITY |
|---------|-------------|-------------------|
| LTO-1 to LTO-7 | n/a | n/a |
| LTO-8 | 78h (i.e., Ultrium 7 DATA volume) | 5Ch (i.e., "U-732" density) or 5Dh (i.e., "U-832M" density) |
| LTO-8 | 7Ch (i.e., Ultrium 7 WORM volume) | 5Ch (i.e., "U-732" density) |
| LTO-8 | 88h (i.e., Ultrium 8 DATA volume) | 5Eh (i.e., "U-832" density) |
| LTO-8 | 8Ch (i.e., Ultrium 8 WORM volume) | 5Eh (i.e., "U-832" density) |

> a - See DENSITY NAME in table 112, Density information LTO-7 and later, on page 142.
> b - Default.
