# 6.4.33 LP 3Eh: Engineering Use

See 5.2.8--LOG SELECT - 4Ch and see 5.2.9--LOG SENSE - 4Dh for directions on how to use this page. This page is for engineering use only and is not included in the list of supported pages (i.e., page code 00h). As such, the counters on this page are not intended for general use and are not shown in this document.

# 6.4.34 LP 3Eh[3Ch]: Drive Control Statistics

See LOG SELECT - 4Ch (see 5.2.8 on page 94) and LOG SENSE - 4Dh (see 5.2.9 on page 94) for directions on how to use this page. This page is for special drive control mode statistics. It is not included in the list of supported pages (page code 00h) and is not intended for general use. This page is not shown in this document.

> **NOTE 43:** This page was log page 3Ch in Ultrium 2, but was moved to log page 3E[3C] for Ultrium 2 HH and newer generations (eServer support with its log page 3C was added).
