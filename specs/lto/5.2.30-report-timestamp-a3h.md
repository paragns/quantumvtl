# 5.2.30 REPORT TIMESTAMP - A3h[0Fh]

The REPORT TIMESTAMP command (see table 129) is defined in SPC (see the latest version at
http://www.t10.org/drafts.htm#SCSI3_CMNDSETS). This clause decribes the specific implementation.

```
                                     Table 129 — REPORT TIMESTAMP CDB

                                                                Bit
        Byte
                      7          6           5            4              3            2             1       0
         0                                           OPERATION CODE (A3h)

         1                    Reserved                                       SERVICE ACTION (0Fh)

         2
                                                              Reserved
         5
         6          (MSB)
                                                      ALLOCATION LENGTH
         9                                                                                                (LSB)
         10                                                   Reserved
         11                                        Control Byte (see 5.1.2.3)
```

The following parameters apply:
- **SERVICE ACTION**: 0Fh
- **ALLOCATION LENGTH**: The number of bytes that have been allocated for the returned parameter data.

## 5.2.30.1 REPORT TIMESTAMP parameter data

The layout of the parameter data returned by the REPORT TIMESTAMP command is shown in table 130.

```
                            Table 130 — REPORT TIMESTAMP Timestamp Descriptor

                                                                   Bit
        Byte
                      7           6           5             4               3         2          1             0
         0          (MSB)
                                              TIMESTAMP PARAMETER DATA LENGTH (0Ah)
         1                                                                                                   (LSB)
         2                                 Reserved                                       TIMESTAMP ORIGIN

         3                                                      Reserved
         4          (MSB)
                                                                TIMESTAMP
         9                                                                                                   (LSB)
         10
                                                                Reserved
         11
```

The parameters are defined as follows:

| Byte | Description |
|------|-------------|
| 0-1 | TIMESTAMP PARAMETER DATA LENGTH: 0Ah |
| 2 | Bit 7-3: Reserved; Bit 2-0: TIMESTAMP ORIGIN: Device Clocks (see 4.17 on page 61) defines the TIMESTAMP ORIGIN |
| 3 | Reserved |
| 4-9 | TIMESTAMP: Device Clocks (see 4.17 on page 61) defines the timestamp. |
| 10-11 | Reserved |
