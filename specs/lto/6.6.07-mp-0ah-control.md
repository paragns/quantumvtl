# 6.6.7 MP 0Ah: Control

See 5.2.10 for how to set these parameters and 5.2.11 for how to read these parameters. Mode Page Behaviors (see 4.7 on page 40) describes this device's non-standard behaviors related to mode parameters.

The mode page policy for this page is described in 4.7.1.

```
Table 335 — MP 0Ah Control mode page

Byte

  3               QUEUE ALGORITHM MODIFIER              Reserved        QERR    DQUE
  4       Reserved     RAC                Reserved               RAERP  UAAERP  EAERP

  5                             Reserved                            AUTOLOAD MODE

  6        (MSB)
                                 READY AEN HOLDOFF PERIOD
  7                                                                            (LSB)

 10        (MSB)
                                        Reserved

```

- **Byte 0**:
  - Bit 7: **PS** (Parameter Saveable): 0b. Ignored in MODE SELECT. See 4.7.2.2.
  - Bit 6: **SPF**: 0b
  - Bits 5-0: **Page Code**: 0Ah
- **Byte 1**: Page Length: 0Ah
- **Byte 2**:
  - Bits 7-5: **TST** (Task Set Type): 000b (non-changeable)
  - Bits 4-2: Reserved
  - Bit 1: **GLTSD** (Global Logging Target Save Disable): 0b (non-changeable)
  - Bit 0: **RLEC** (Report Log Exception Condition): 0b (non-changeable)
- **Byte 3**:
  - Bits 7-4: **QUEUE ALGORITHM MODIFIER**: 0000b (non-changeable)
  - Bits 3-2: Reserved
  - Bit 1: **QERR** (Queue Error): 0b (non-changeable)
  - Bit 0: **DQUE** (Disable Queuing): 1b (non-changeable)
- **Byte 4**:
  - Bit 7: Reserved
  - Bit 6: **RAC** (Report A Check): 0b (non-changeable)
  - Bits 5-3: Reserved
  - Bit 2: **RAERP** (Ready Asynchronous Event Reporting): 0b (non-changeable)
  - Bit 1: **UAAERP** (Unit Attention Asynchronous Event Reporting): 0b (non-changeable)
  - Bit 0: **EAERP** (Error Asynchronous Event Reporting): 0b (non-changeable)
- **Byte 5**:
  - Bits 7-3: Reserved
  - Bits 2-0: **AUTOLOAD MODE**: 000b (changeable)
    - Some medium changers may silently override AUTOLOAD MODE.

    | Value | Description |
    |-------|-------------|
    | 000b | Medium is loaded for full access. |
    | 001b | Medium is loaded for MAM access only (i.e., load hold position). |
    | 010b | Medium is not allowed to be loaded. |
    | others | Reserved. |

    > **WARNING**: Setting AUTOLOAD MODE to a value of 010b may cause problems for medium changers (e.g., fail move medium commands, interference with pickers, etc.)

- **Bytes 6-7**: READY AEN HOLDOFF PERIOD: 0000h (non-changeable)
  - If AEN is disabled (Byte 4 bit 2 = 0b), this field is not meaningful.
- **Bytes 8-9**: BUSY TIME-OUT PERIOD: FFFFh (non-changeable)
- **Byte 10**: Reserved
- **Byte 11**: Reserved
